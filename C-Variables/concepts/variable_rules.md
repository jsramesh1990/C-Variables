# C Variable Rules and Concepts

## 1. Variable Declaration Rules

### Valid Rules:
- Must begin with a letter or underscore
- Can contain letters, digits, and underscores
- Case-sensitive
- Cannot be a C keyword
- Should be meaningful and descriptive

### Examples:
```c
int age;          // Valid
float _temp;      // Valid (starts with underscore)
double salary2;   // Valid (contains digit)
char first_name;  // Valid (uses underscore)
